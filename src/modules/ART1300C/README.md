# ART1300C Drawing

## Course Description
Basic problems in freehand drawing, including perspective, still-life and landscape. Emphasis is on developing a sense of structure through line, form and texture.

## Course Structure
Total Assignments: 14 (140 points)
All assignments are worth 10 points each

### Module 1: Foundation Skills (Weeks 1-4)
**Learning Objectives**
* Master basic drawing techniques
* Understand perspective and proportion
* Develop observational skills

**Assignments (4 assignments, 40 points)** 1. Value House Drawing Assignment 2. The Big Challenge 3. Viewfinder 4. The Mondrian Line Assignment

### Module 2: Form and Space (Weeks 5-8)
**Learning Objectives**
* Master light and shadow
* Develop spatial relationships
* Create depth through value

**Assignments (3 assignments, 30 points)** 5. Light and Shadow with Charcoal Drawing 6. All over Drawing 7. Out of Focus Monet Drawing

### Module 3: Advanced Techniques (Weeks 9-11)
**Learning Objectives**
* Master complex compositions
* Develop personal style
* Understand figure-ground relationships

**Assignments (3 assignments, 30 points)** 8. Perspective 9. Proportion 10. Ten Flag Drawings

### Module 4: Portfolio Development (Weeks 12-16)
**Learning Objectives**
* Create portfolio-quality work
* Refine technical skills
* Demonstrate mastery

**Assignments (4 assignments, 40 points)** 11. Frottage Drawing 12. Ellipses Drawing 13. Straight-line Drawing 14. Searching Line Drawing
